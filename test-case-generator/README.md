# Automated Test Case Generator

AI-powered system that generates programming test cases from problem descriptions.

## What It Does

**Input**: Problem description with constraints  
**Output**: JSON with test cases (normal, edge, boundary)

## Usage

1. Enter problem description
2. Select model (Qwen 1.5B recommended)
3. Click "Generate Test Cases"
4. Copy JSON output

## Example

**Input**: Two Sum problem  
**Output**: Valid test cases in JSON format
